DataManager Usage Instructions

Installation

composer require fereydooni/data-manager

Configuration

If using Laravel, publish the config file:

php artisan vendor:publish --provider="DataManager\Providers\DataManagerServiceProvider" --tag=config
Tip: You can customize default format, chunk size, logging, and error handling in config/data-manager.php.

Core Features

  • Import and export data in multiple formats: CSV, JSON, XML, SQL, Excel, database models, Spatie Data objects
  • Custom importers/exporters via interfaces
  • Data validation and transformation hooks
  • Batch processing for large datasets
  • Logging, error handling, and progress tracking utilities
  • Event hooks for import/export lifecycle
  • Extensible and configurable

Custom Transformers

Advanced! You can apply a custom transformer to each item during import/export.
Pass a callable or a class with a transform( $item) method.

Example: Using a Closure

use DataManager\Facades\DataManager;

$rows = DataManager::import('csv', 'file.csv', function($item) {
    $item['name'] = strtoupper($item['name']);
    return $item;
});

DataManager::export('json', $data, 'file.json', function($item) {
    unset($item['password']);
    return $item;
});

Example: Using a Transformer Class

use DataManager\Facades\DataManager;

class MyTransformer {
    public function transform($item) {
        $item['email'] = strtolower($item['email']);
        return $item;
    }
}

DataManager::setTransformer(new MyTransformer());
$rows = DataManager::import('json', 'file.json');

Validation

Advanced! You can validate each item during import using a callable or a class with a validate($item) method. Invalid items are skipped and collected in an $errors array.

Example: Using a Closure

use DataManager\Facades\DataManager;

$errors = [];
$rows = iterator_to_array(DataManager::import('csv', 'file.csv', null, function($item) {
    return filter_var($item['email'], FILTER_VALIDATE_EMAIL);
}, $errors));

// $errors contains invalid rows

Example: Using a Validator Class

use DataManager\Facades\DataManager;

class MyValidator {
    public function validate($item) {
        return !empty($item['name']) && is_numeric($item['age']);
    }
}

$errors = [];
DataManager::setValidator(new MyValidator());
$rows = iterator_to_array(DataManager::import('json', 'file.json', null, null, $errors));

Event Hooks

Advanced! Listen for import/export lifecycle events: import.before, import.after, import.row, import.error, export.before, export.after, export.row, export.error.

Example: Listening for Events

use DataManager\Utils\EventDispatcher;

EventDispatcher::listen('import.before', function($type, $source) {
    // Log or handle before import
});

EventDispatcher::listen('import.row', function($item) {
    // Handle each imported row
});

EventDispatcher::listen('import.error', function($row) {
    // Handle validation error
});

EventDispatcher::listen('import.after', function($type, $source) {
    // After import
});

Example: Export Events

EventDispatcher::listen('export.before', function($type, $target) {
    // Before export
});
EventDispatcher::listen('export.row', function($item) {
    // Each exported row
});
EventDispatcher::listen('export.error', function($exception) {
    // Handle export error
});
EventDispatcher::listen('export.after', function($type, $target) {
    // After export
});

Batch & Chunk Processing

Advanced! Process large files in chunks by passing a chunkSize parameter to import or export. Each yielded value will be an array of items.

Example: Import in Chunks

use DataManager\Facades\DataManager;

foreach (DataManager::import('csv', 'file.csv', null, null, $errors, 100) as $chunk) {
    // $chunk is an array of up to 100 rows
    // Process chunk
}

Example: Export in Chunks

use DataManager\Facades\DataManager;

DataManager::export('json', $data, 'file.json', null, 500); // Exports in chunks of 500

Asynchronous/Queued Operations

Advanced! You can run import/export jobs asynchronously using Laravel queues. Make sure your queue worker is running.

Example: Queue an Import Job

use DataManager\Facades\DataManager;

DataManager::queueImport('csv', 'file.csv', $transformer, $validator, 100);

Example: Queue an Export Job

use DataManager\Facades\DataManager;

DataManager::queueExport('json', $data, 'file.json', $transformer, 500);
Note: You must configure Laravel's queue system and run php artisan queue:work for jobs to be processed.

Progress Reporting & Notifications

Advanced! Listen for import.progress and export.progress events to track progress. You can send notifications (email, Slack, etc.) in your event handlers.

Example: Progress Listener

use DataManager\Utils\EventDispatcher;

EventDispatcher::listen('import.progress', function($count, $total, $percent) {
    echo "Imported $count of $total (" . round($percent) . "%)\n";
});

EventDispatcher::listen('export.progress', function($count, $total, $percent) {
    // Send progress to Slack, email, etc.
});
Tip: Use Laravel's notification system in your event handler to send emails, Slack messages, or other notifications.

Utilities

  • Logger: DataManager\Utils\Logger::log('message');
  • Exception: throw new DataManager\Exceptions\DataManagerException('error');
  • Progress Tracker: 
    $tracker = new DataManager\Utils\ProgressTracker($total);
foreach ($items as $item) {
    // ...
    $tracker->advance();
}
$tracker->finish();

Extending

  • Implement ImporterInterface or ExporterInterface for custom formats.
  • Use DataManager\Utils\EventDispatcher for event hooks:
use DataManager\Utils\EventDispatcher;
EventDispatcher::listen('import.started', function ($source) {
    // ...
});
EventDispatcher::dispatch('import.started', $source);

Configuration Options

  • default_format: Default format for import/export (e.g., csv)
  • chunk_size: Batch size for processing
  • logging: Enable/disable logging
  • error_handling: Log errors, throw exceptions, etc.
